# Feedbacker
A Discord bot to allow people to make polls and surveys for their communities.

## Installation
- [Link for the bot](https://discord.com/oauth2/authorize?client_id=1493267057846915218&permissions=670149167120&integration_type=0&scope=bot+applications.commands)
- Add the bot to the server
- Set it Up with the `/setup <approved_role> <approved_channel>` command (I recommend making another role for Poll/survey making)
- In the approved channel, use the `/poll` or `/survey` commands as told in the options and their descriptions
- Or just [watch this video](https://youtu.be/FZ4xhsqGheo)

## Commands
- `/poll <title> <optionCount: 1-5> <option 1> [option 2-5]`
- `/survey <title> <description> <field 1> [field 2-5]`
- `/setup <approved_role> <approved_channel>` 

(options marked with "<>" are required, "[]" are optional)
